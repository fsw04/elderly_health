## MVP关键决策（基线）
- 用户登录：微信openid登录 + 绑定手机号
- 家属：两阶段绑定（申请→老人确认），家属只读查看报告
- 体检报告：规则阈值（非AI）自动生成
- 未体检提醒：固定周期（3个月/6个月）→ 医生待办 → 医生触达记录
- 血糖编码：`GLU_FASTING`（空腹）与 `GLU_RANDOM`（随机）两个code
- ECG策略：分片不完整则“完全不出报告”
- MQTT安全：TLS + HMAC-SHA256签名（含topic与sentAt防重放）
